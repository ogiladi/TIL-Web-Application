Did You Know?

Learn something new at a click of a button


This webpage uses Django to run python and uses the PRAW (Python Reddit API Wrapper) to extract content from Reddit's TIL (today I learned) subreddit. The app takes threads and then stores them on a SQLite database. When a user clicks a button, a random thread from the database is selected. To see the full code, see my GitHub page .
